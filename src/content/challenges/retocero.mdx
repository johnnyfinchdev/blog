# ¡Hola Developers!

  > En este reto pondrás a prueba tus habilidades con **arrays** y **funciones** en JavaScript.
  Tu misión será crear una función que reciba un arreglo de números y devuelva **solo aquellos que sean números primos**, ordenados de menor a mayor.

{/* Seccion Enunciado */}
## Enunciado

  ```json title="index.js"
  /* 
  Crea una función llamada `filtrarPrimos` que reciba un arreglo de números enteros.
  Debe devolver un nuevo arreglo con todos los números primos del arreglo original, 
  ordenados de menor a mayor.

  Ejemplo:
  Entrada: [4, 7, 10, 11, 13, 2, 9]
  Salida: [2, 7, 11, 13]
  */
  ```

    {/* Solución */}
    <details>
      <summary>Solución</summary>
        Aquí tienes una posible implementación de la función:

        ```javascript title="solucion.js"
        function esPrimo(num) {
          if (num <= 1) return false;
          for (let i = 2; i <= Math.sqrt(num); i++) {
            if (num % i === 0) return false;
          }
          return true;
        }

        function filtrarPrimos(arr) {
          return arr
            .filter(esPrimo)
            .sort((a, b) => a - b);
        }

        // Ejemplo de uso:
        console.log(filtrarPrimos([4, 7, 10, 11, 13, 2, 9]));
        // Resultado esperado: [2, 7, 11, 13]
        ```
    </details>
