{/* Seccion Intro */}
# ¡Hola Developers!

  > En este reto pondrás a prueba tus habilidades con **strings** y **lógica** en JavaScript.
  Tu misión será crear una función que determine si una palabra o frase es un **palíndromo** (se lee igual al derecho que al revés).

{/* Seccion Enunciado */}
## Enunciado

  ```json title="index.js"
  /* 
  Crea una función llamada `esPalindromo` que reciba una cadena de texto.
  Debe devolver true si la cadena es un palíndromo y false si no lo es.
  Ignora espacios, tildes y diferencias entre mayúsculas y minúsculas.

  Ejemplo:
  Entrada: "Anita lava la tina"
  Salida: true
  
  Entrada: "Hola"
  Salida: false
  */
  ```

    {/* Solución */}
    <details>
      <summary>Solución</summary>
        Aquí tienes una posible implementación de la función:

        ```javascript title="solucion.js"
        function esPalindromo(cadena) {
          const limpia = cadena.toLowerCase().replace(/\s/g, '');
          const invertida = limpia.split('').reverse().join('');
          return limpia === invertida;
        }

        // Ejemplo de uso:
        console.log(esPalindromo("Anita lava la tina")); // true
        console.log(esPalindromo("Hola"));               // false
        ```
    </details>
