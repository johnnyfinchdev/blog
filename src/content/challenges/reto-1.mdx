{/* Seccion Intro */}
# ¡Hola Developers!
  > En este reto pondrás a prueba tus habilidades con **strings** en JavaScript.
  Tu misión será crear una función que reciba una cadena de texto y devuelva la cadena **invertida**.

{/* Seccion Enunciado */}
## Enunciado

  ```json title="index.js"
  /* 
  Crea una función llamada `invertirCadena` que reciba una cadena de texto.
  Debe devolver la misma cadena pero con los caracteres en orden inverso.

  Ejemplo:
  Entrada: "hola mundo"
  Salida: "odnum aloh"
  */
  ```

      <summary>Pista 1</summary>
      Piensa en cómo convertir una cadena en un array de caracteres.

      ```javascript title="pista1.js"
      const chars = cadena.split('');
      ```

    {/* Pista 2 */}
    <details>
      <summary>Pista 2</summary>
      Una vez tengas el array, puedes invertirlo con el método **reverse()**.


      ```javascript title="pista2.js"
      chars.reverse();
      ```
    </details>

<br/>



    {/* Pista 3 */}
    <details>
      <summary>Pista 3</summary>
      Finalmente, une los caracteres de nuevo con **join()**.

      ```javascript title="pista3.js"
      chars.join('');
      ```
    </details>
<br/>


    {/* Pista 4 */}
    <details>
      <summary>Pista 4</summary>
      Puedes encadenar los tres métodos en una sola línea.


      ```javascript title="pista4.js"
      cadena.split('').reverse().join('');
      ```

    {/* Solución */}
    <details>
      <summary>Solución</summary>
        Aquí tienes una posible implementación de la función:

        ```javascript title="solucion.js"
        function invertirCadena(cadena) {
          return cadena.split('').reverse().join('');
        }



        // Ejemplo de uso:
        console.log(invertirCadena("hola mundo"));
        // Resultado esperado: "odnum aloh"
        ```

    </details>
