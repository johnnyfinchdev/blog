{/* Seccion Intro */}
# ¡Hola Developers!

  > En este reto pondrás a prueba tus habilidades con **strings** e **iteración** en JavaScript.
  Tu misión será crear una función que reciba una cadena de texto y devuelva el **número de vocales** que contiene.

{/* Seccion Enunciado */}
## Enunciado

  ```json title="index.js"
  /* 
  Crea una función llamada `contarVocales` que reciba una cadena de texto.
  Debe devolver el número total de vocales (a, e, i, o, u) que contiene la cadena,
  sin distinguir entre mayúsculas y minúsculas.

  Ejemplo:
  Entrada: "Hola Mundo"
  Salida: 4
  */
  ```

    <details>
      <summary>Solución</summary>
        Aquí tienes una posible implementación de la función:

        ```javascript title="solucion.js"
        function contarVocales(cadena) {
          const vocales = 'aeiou';
          return cadena.toLowerCase().split('').filter(c => vocales.includes(c)).length;
        }

        // Ejemplo de uso:
        console.log(contarVocales("Hola Mundo"));
        // Resultado esperado: 4
        ```
